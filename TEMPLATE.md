### `interface`
This file will be by the system to create your website
*   `e(__SITE_TITLE__)`: The website title, this will be used to show the website page title.
##### CSS Designs
Allowing Users To Change The colors on your template, use the following anchor tag `e(__DESIGN__COLOR_<name>__)`. 

###### Examples

*   `e(__DESIGN_COLOR_BACKGROUND__)` `[Optional]`: This is an optional feature where the css home page designs can be changed by the users. 
*   `e(__DESIGN_COLOR_TEXT__)` `[Optional]`: The design of the font color.
*   `e(__DESIGN_COLOR_PRIMARY__)` `[Optional]`: The design of the font color of the primary.



*   `e(__SITE_TITLE__)`: The website title, this will be used to show the website page title.
